# nodejs-web-api-express-redis-del-key

## Description
A basic web api looksup, deletes a key,
and returns true/false if key exists.

## Tech stack
- nodejs

## Docker stack
- node:latest
- redis:latest

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
